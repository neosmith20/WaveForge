Windows: One-Time Driver Setup

Windows ships with a default STM32 DFU driver that blocks WebUSB access. You need to swap it for WinUSB using a free tool called Zadig. This is a one-time step — future flashes will work without repeating it.

  1. Put your radio into DFU mode using the instructions in Step 2 below, then come back here.
  2. Download and run Zadig.
  3. In Zadig, click Options → List All Devices.
  4. Select Digital Radio in USB mode from the device dropdown.
  5. Use the up/down arrows to select WinUSB (v6.1.7600.16385), then click Replace Driver.
  6. Once the driver installs, return here and click Connect in Step 2.

Connect Your Radio

Put your radio in flash mode first

  1. Make sure the radio is connected to your computer via USB.
  2. Turn the radio off.
  3. Hold PTT + SK1 (the two side buttons) simultaneously.
  4. While holding both buttons, turn the radio on.
  5. The LED will flash red/green — that confirms the radio is in DFU mode.

Provide Donor Firmware

What is a donor file?

ClearDMR uses a vocoder — the audio codec that encodes and decodes DMR voice. Due to licensing restrictions, this codec cannot be bundled with ClearDMR. Instead, the flasher extracts it from your radio's original factory firmware, which you already own a license to use.

Obtain the original firmware .bin for your specific radio model from the manufacturer's website, then select it below. The file never leaves your browser. If you need a known-good donor file, you can download one here.
